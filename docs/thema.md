## 1. プロダクトコンセプト

* **一言**

  * 「Mac のトレイから呼び出す、ストリーミング翻訳レーン」
* **役割**

  * どこでテキストを選択していても、⌘J 一発で

    * 日本語 ⇄ 英語を即ストリーミング翻訳
  * 結果は軽い表示（通知 or シンプル 2ペイン UI）
* **方針**

  * ローカル常駐 + OpenAI API だけ
  * 設定は「APIキーとモデル」程度のミニマム
  * OSS / brew 前提、個人でも組み込みやすいユーティリティ

---

## 2. 名前と世界観（tray + lingo）

* **tray**

  * Mac のメニューバー／トレイ領域に静かに常駐
  * 「いつでもそこにいるけど、主役にはならない」裏方ツール
* **lingo**

  * 言語・スラング・文脈をさっと橋渡しする存在
  * 辞書ほど重くない、「ちょっと聞く」感じの会話相手

→ **「画面の端にいる、小さな通訳トレイ」** くらいのニュアンス。

---

## 3. UI / UX コンセプト

* **操作**

  * 基本 UX：選択 → ⌘J
  * 選択テキストを自動コピー → 翻訳 → 結果表示
* **表示**

  * 通知 or 小さなウィンドウで要点だけ
  * 詳しく見たいときは 2ペイン画面：

    * 左：原文（そのまま）
    * 右：翻訳（ストリーミングで伸びていく）
* **性格**

  * 説明過多にしない（訳文だけ返す）
  * 完了を待たず、流れてくるテキストをそのまま見せる

---

## 4. ビジュアルコンセプト（アイコン・カラー）

* **全体トーン**

  * テーマ：**トレイ + ことば**
  * シンプル・フラット寄り、余計な装飾なし
* **形**

  * 「トレイ」を連想させる、横長のプレート / 角丸長方形
  * その中に：

    * 左に “A”
    * 右に “あ”
    * もしくは 2つのバブル / 吹き出しが乗っているイメージ
* **カラー**

  * ベース：グレー〜ダークグレー（メニューバーと相性良い無彩色）
  * アクセント：**うっすら赤系**

    * トレイの縁だけ薄い赤
    * もしくは “A / あ” の片方を赤系アクセント
  * 派手な真っ赤ではなく、**ワインレッド / サーモン寄りの落ち着いた赤** を想定
* **スタイル**

  * 影は薄く、少しだけ奥行きが分かる程度
  * ベクター感のあるフラット寄り
  * 「ツール感＞キャラクター感」

## 5. 技術・設計ポリシー

* **プラットフォーム**

  * Mac 専用（hotkey + tray 前提）
* **アーキテクチャ**

  * Rust 単一バイナリ

  * OpenAI Streaming 前提の API クライアント
* **配布**

  * GitHub + Homebrew tap
  * OSS（MIT or Apache-2.0）
